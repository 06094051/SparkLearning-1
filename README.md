# 1.环境 #
SparkLearning是在spark上运行的，spark搭建等请见spark官网或其他网站。
SparkLearning运行环境:  
Spark-1.5.2  
eclipse-4.3.2  
scala-2.10.4  
jdk1.7  
idea 15.04  
spark-assembly-1.5.2-hadoop2.6.0.jar(下载地址: http://pan.baidu.com/s/1hrSxiDI)
